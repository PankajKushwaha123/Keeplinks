KeepLinks- Link Protecting Script

Version 1.1
KeepLinks – Link Protecting Script is written mainly in PHP with a creative HTML5 Front-end and Back-end User-Interface. This script is designed to keep your links at a more private and safer place and allows you to prevent them from being listed by the Search Engines. All free as well as registered users and allowed to protect their links with the script but only the registered users are able to manage the links using an ultra-modern User Panel and along-with this, they can also earn  by a ‘Pay-Per-View’ program for which the payout limits and per view amounts are decided by the Administrator only.

Requirements

  1. PHP 5.6 or 7.x
  2. MySQL 5.x or above
  3. Some patience and time
  4. Remember that most of the features and fuctions may not work perfectly on localhost. So you should upload this script to a webhost and work accordingly.

Installation

This script is powered by a quick web-based installer for an easy and convenient installation of the script without any errors or hard efforts. Follow the below steps for installation.

  1. Upload and Extract the main.zip file to your host/server.
  2. Point your Browser to http://yoursite.com/install.php
  3. Select the Install option to proceed with installation procedure..
  4. Follow the on-screen instructions carefully.
  5. After successful installation, please ensure that whether install.php has been deleted automatically or not. If not deleted delete it manually.
  6. Now, point your browser towards http://yoursite.com/admin and login using the admin credentials that you set-up while installation.
  7. Now on the admin dashboard you can get stats for your site and configure it according to your needs.
  8. Congrats! Now you've successfully installed the KeepLinks script.

Upgradation

If you already have installed previous version of KeepLinks script, then you may want to proceed with the upgradation of KeepLinks to new version.

The upgradation process is also very much similar to that of installation.

  1. Upload and Extract the main.zip file to your host/server.
  2. Point your Browser to http://yoursite.com/install.php
  3. Select the Upgrade option to proceed with upgradation procedure.
  4. Follow the on-screen instructions carefully.
  5. After successful upgradation, please ensure that whether install.php has been deleted automatically or not. If not deleted delete it manually.
  6. Congrats! Now you've successfully updated your KeepLinks script to the latest version.
